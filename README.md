# Conquer Game

An isometric ARPG inspired by Conquer Online, built with Godot 4.6.

## About

Venture into a world of combat and conquest! This game captures the classic feel of Conquer Online with:
- Isometric click-to-move controls with smooth acceleration
- Dynamic combat system with normal attacks and AoE scatter skill
- Progressive difficulty zones with 7 monster types
- Visual polish with particle effects and sound
- Monster UI with name labels and health bars

## Game Controls

- **Left Click**: Move to position / Select target
- **Ctrl + Left Click**: Jump to position
- **Right Click**: Scatter skill (AoE attack)
- **Mouse Wheel**: Zoom in/out
- **+/- Keys**: Zoom in/out
- **0**: Reset zoom

## Combat System

- **Normal Attack**: Click on a monster to target, auto-attacks when in range
- **Damage**: 80-120% of Attack Power with 15% critical hit chance (1.5x damage)
- **Scatter Skill**: Right-click to fire arrows in a cone, hits multiple targets
- **Monsters**: 7 types with increasing difficulty from center

## Technical Details

- **Engine**: Godot 4.6
- **Rendering**: 2D isometric with geometric shapes
- **Max Entities**: 1500 monsters with optimized performance
